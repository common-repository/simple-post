Display a simpler Write Post screen.

== Description ==

Simple Post gives the ability to create a simplified write Post screen. Options are used to set the default category, ping status, and comment status. The post title can also be set using a combination of text and date/time. A couple of more options can be set to display the Tags and the Editor toolbar.

== Installation ==

1. Upload `simple-post.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin options at Options->Simple Post in the WordPress admin
